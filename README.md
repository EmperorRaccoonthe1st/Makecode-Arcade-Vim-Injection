# Usage
The script found within the main.js file is designed for use by browser extensions like ViolentMonkey to inject into the arcade.makecode.com webpage.
It patches in Vim support to the monaco instance that the makecode editor uses for Javascript and Python.
It was vibe coded with the help of Gemini.

# Constraints
I have only tested the script on Firefox using ViolentMonkey, so expect errors on any other browser/platform.
